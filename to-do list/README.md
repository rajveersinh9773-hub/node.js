#  To-Do List App

A simple To-Do List application built using **Node.js**, **Express.js**, **EJS**, and **CSS**. Users can add and delete tasks through a clean and responsive interface.

##  Features

- Add new tasks
- Delete existing tasks
- Server-side rendering using EJS
- Responsive and modern UI
- Uses Express middleware for form handling

##  Technologies Used

- Node.js
- Express.js
- EJS
- HTML5
- CSS3

##  Installation

1. Clone the repository

```bash
git clone <https://github.com/rajveersinh-jadeja/node.js/tree/main/to-do%20list>
```

2. Navigate to the project folder

```bash
cd to-do-list
```

3. Install dependencies

```bash
npm install
```

4. Start the application



```bash
node app.js
```


##  Open in Browser

```
http://localhost:3000
```
 
##  How It Works

- The home page displays all tasks.
- Users enter a task and click the **Add** button.
- The task is stored in an array on the server.
- Clicking the **Delete** button removes the selected task.
- The page reloads automatically after each action.
